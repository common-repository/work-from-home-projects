Exposes your users to paying work-from-home projects aggregated from all across the web based on their skills and your site content.

== Description ==

Projects are location independent and can be performed from home in exchange for payment.
The widget can be completely customized to enhance your site's design and offer projects relevant to your user's set of skills and interest.

== Installation ==

1. Upload `donanza.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Drag the widget to the prefered location on your side bar
4. Use the configuration screen to customize the widget theme and colors to your site.
